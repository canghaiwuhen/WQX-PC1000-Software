《复数向量表达式运算》
!!!!ATTENTION:由于程序涉及大量的浮点数运算和递归调用，所以的生成代码很长，500多行的程序，竟然老是说代码超长，没办法，热线帮助省了又省，不少话没有写进去，所以请一定要仔细阅读以下内容，最好下载到文曲星里。

功能：本程序用于计算带复数向量的表达式的值，支持括号、幂运算符，累加减，简化输入，界面友好，操作简单方便，可帮助大家求解工程算式，当然，也可以做实数范围内的运算，嘻嘻！

一、!!!!!注意事项：
1、本程序支持括号，幂运算符，可省略乘号：j(2+3.4L30')L3.14/2.2L1.57=j*1*(2+3.4L30')*1L3.14/(2.2L1.57)
由于可省略乘号，所以规定：
乘号或除号优先级 < 省略乘号的计算优先级 < 幂预算符优先级
如：2*j/3jj(6+4L30')^.5(2-1*1)=2*j/{3*j*j*[ (6+4L30')^( .5*(2-1*1) ) ] }
2、!!进行幂运算时，如果数据太大，可能会发生溢出错误，显示：multiply overflow，这时应当按下网络键重新进入，否则要死机。
3、计算完成后向量角度显示在消息栏中。如有表达式语法错误，就不显结果，只显示出错信息，这时您可修改错误而不用重新输入，方便吧！
4、对于向量，最后结果输出的角度、弧度都是换算后在[-pi，pi]之间的，即[-180'，180']。
5、!!!向量的角度表示法：10L12',12.3L(-30)' //不可写成12.3L-30或12.3L(-30')!!.前者认为是12.3L0-30,后者判为语法错误
                         

二、界面介绍：
1、第一行是窗口标题栏，激活的窗口显示浅色，后台窗口显示深色，目前版本只有两个窗口：MAIN和HELP。
2、标题栏右边是当前模式显示区，从左到右依次是：当前虚数符号或向量符号， 插入/覆盖模式、积存器状态。
2、第二至第五行是表达式输入区，表达适应少于64字节。
3、第六至第九行是结果输出区，第六行是复数形式，第七行是向量形式，方向用弧度表示。
4、第十行是消息栏，显示当前消息。

三、输入表达式：
1、F1键切换插入、覆盖模式；
2、F2键删除一个字符；
3、Z键输入“(”；
4、SHIFT+Z键输入“)”；
5、D键输入“^”;
6、K键在表达式首端加减一个‘-’；
7、“求助”键显示帮助。
8、SHITF+C输入角度符号“'”(因为文曲星小字体的ASCII码不是标准的ASCII，其中没有角度符号，所以我选择了‘'’。
9、左右键将光标移动一个字符的位置；
10、翻页键将光标移到表达式首端或末端；
11、数字键同；
12、加减乘除同键盘标注。
13、最近新加的功能：SHIFT+右键将上次计算的结果存入寄存器，在屏幕右上方显示一个M，SHIFT+P清除积存器值，输入键将之写到表达式中。但要注意，当M有值时，再按SHIFT+M，即是累加功能！！将最近的结果值累加到M。当然，SHIFT+下页键是累减，从M中减掉最近的结果值。

总之输入方式几乎完全和键盘标注的一样，不用去记，试试便知。输入完成后按=，只要没有语法错误，一切搞定 。

四、其他
1、!!!!!对了，还要告诉各位大侠，对于pow()函数：当底数<0时，如果指数是正整数，是有意义的，但是pow()却只返回指数的平方。
所以使用这个函数时要单独针对底数<0的情况进行修正!!!
